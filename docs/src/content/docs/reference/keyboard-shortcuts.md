## Global

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + ,` | Open Settings |
| `Cmd/Ctrl + R` | Refresh current resource list |
| `Escape` | Close open panel / dismiss modal |

## Resource tables

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Move selection up / down |
| `Enter` | Open detail panel for selected row |
| `Delete` / `Backspace` | Delete selected resource (shows confirmation) |

## YAML editor

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + S` | Apply YAML to cluster |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |
| `Cmd/Ctrl + /` | Toggle line comment |

## Log viewer

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + F` | Search within log output |
| `Cmd/Ctrl + End` | Jump to latest log line |
| `Cmd/Ctrl + Home` | Jump to beginning of log buffer |

:::note
Keyboard shortcut support is actively expanding. Check the [GitHub releases](https://github.com/DEVtheOPS/kore/releases) for additions in each version.
:::
